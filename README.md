Developed a fully functional Android e-commerce application for browsing products across multiple categories.
Implemented cart management with quantity adjustment, dynamic pricing, and a structured checkout flow.
Enabled order status tracking through complete lifecycle states: Ordered → Confirmed → Shipped → Delivered → Cancelled.
Designed modules for order history and multi-address management, supporting flexible delivery options.
Added Profile Management allowing users to update personal information, profile image, and saved delivery addresses.
Developed an Admin Application to add, update, and manage product inventory and categories in real-time.
Structured the application using MVVM architecture to ensure maintainability, scalability, and efficient state management.
Tech Stack: Kotlin | Android SDK | MVVM Architecture | Firebase Firestore | Firebase Authentication | Firebase Cloud Storage | Glide
